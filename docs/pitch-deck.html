How it works

Stripe collects. Morph settles. No SWIFT, no correspondent banks.

The webhook is the only thing that crosses

USD never touches the chain. When Stripe says “paid,” NestJS independently signs an ERC-20 USDC.transfer with viem and broadcasts to Morph. The chain validates the signature and includes the tx — that's it.

Every payment is verifiable

We only write morph_tx_hash after waitForTransactionReceipt confirms (~2s). The toast we show the freelancer is never a lie — there's a public Explorer link to prove it.

payment_intent.succeeded → trigger 10s polling fallback (demo-safe) Supabase Realtime status push (next) FX snapshot stored at settlement
